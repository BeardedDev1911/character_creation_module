# Character Creation Module

Учебный проект: модуль создания и тренировки персонажа для RPG-игры.

## Описание

Программа запускается в консоли и предлагает игроку:

- ввести имя персонажа;
- выбрать класс персонажа;
- подтвердить выбор;
- пройти тренировку, используя доступные команды.

В проекте реализованы базовый класс `Character` и три класса персонажей:

- `Warrior` - воитель;
- `Mage` - маг;
- `Healer` - лекарь.

У каждого класса есть свои значения атаки, защиты и специального умения.

## Команды тренировки

Во время тренировки доступны команды:

- `attack` - атаковать противника;
- `defence` - заблокировать атаку противника;
- `special` - применить специальное умение;
- `skip` - завершить тренировку.

## Как запустить

Перейдите в папку проекта:

```bash
cd "Sprint 1/character_creation_module"
```

Запустите файл:

```bash
python main.py
```

Если используется Windows и команда `python` не работает, попробуйте:

```bash
py main.py
```
